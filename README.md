# Abstração de um jogo RPG
Este projeto foi desenvolvido utilizando .Net 8. O mesmo teve como principio colocar em ação os 4 pilares de Orientação a Objetos:
- Herança;
- Encapsulamento;
- Polimorfismo;
- Abstração;
